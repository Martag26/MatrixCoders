# MatrixCoders

MatrixCoders es una plataforma pensada para estudiantes de DAW.  
Un espacio integrado para consultar cursos, hacer ejercicios y practicar flujos reales de desarrollo web.

Está diseñada para facilitar el aprendizaje diario y poder ampliarse con facilidad.

## Tecnologías y lenguajes

- **HTML5**  
  Estructura y contenido de la web.

- **CSS3**  
  Estilos, diseño visual y maquetación.

- **JavaScript**  
  Interactividad y funcionalidades en el cliente.

- **Spring Boot**  
  Backend y lógica del servidor; implementación del patrón MVC.

- **Arquitectura MVC**  
  Organización del proyecto y separación de responsabilidades.

- **PostgreSQL**  
  Base de datos relacional para datos relacionados.

- **Stripe**  
  Simulación e integración de pagos.
