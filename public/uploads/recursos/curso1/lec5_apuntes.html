Apuntes — Selectores CSS

Desarrollo Web · U3 · CSS

CSS (Cascading Style Sheets) define la presentación visual de los documentos HTML. Los selectores indican a qué elementos aplicar los estilos.

Tipos básicos de selectores

/* Selector de etiqueta */
p { color: #333; }

/* Selector de clase */
.boton { background: #6B8F71; }

/* Selector de id */
#cabecera { padding: 20px; }

/* Selector universal */
* { box-sizing: border-box; }

Selectores combinados

• Descendiente: nav a → cualquier <a> dentro de <nav>.
• Hijo directo: nav > a → solo hijos directos.
• Hermano adyacente: h1 + p.
• Hermano general: h1 ~ p.
• Lista: h1, h2, h3 → aplica a varios selectores.

Pseudoclases y pseudoelementos

• :hover, :focus, :active — estados.
• :first-child, :last-child, :nth-child(n) — posición.
• ::before, ::after — añaden contenido virtual.

Especificidad

Cuando varias reglas afectan al mismo elemento gana la de mayor especificidad: !important > inline > id > clase/atributo/pseudoclase > etiqueta. En empate, gana la última declarada.
