Apuntes — Flexbox y Layout

Desarrollo Web · U3 · Maquetación

Flexbox es un sistema de diseño unidimensional que facilita alinear y distribuir el espacio entre elementos, incluso cuando su tamaño es dinámico.

Activar flexbox

.contenedor {
  display: flex;
  /* o display: inline-flex; */
}

Propiedades del contenedor

• flex-direction: row | column | row-reverse | column-reverse.
• justify-content: alineación en el eje principal.
• align-items: alineación en el eje cruzado.
• flex-wrap: permite saltar a la siguiente línea.
• gap: espacio entre hijos.

Propiedades de los hijos

.item {
  flex: 1;                /* equivalente a flex: 1 1 0 */
  align-self: center;     /* sobreescribe align-items */
  order: 2;               /* cambia el orden visual */
}

Ejemplo: navegación responsive

.nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 1rem;
  flex-wrap: wrap;
}

CSS Grid (mención)

Para layouts bidimensionales (filas Y columnas a la vez) usa CSS Grid: display: grid; grid-template-columns: …; grid-template-rows: …;. Combinado con Flexbox cubre el 99% de las maquetaciones modernas.
