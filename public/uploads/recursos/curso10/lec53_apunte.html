Git y GitHub

Configuración y subida de proyectos a GitHub

¿Qué es GitHub?

GitHub es una plataforma web que aloja repositorios Git en la nube. Añade sobre Git: interfaz visual, pull requests, Issues, GitHub Actions (CI/CD), Pages, Projects y mucho más.

Autenticación con SSH (recomendado)

# 1. Generar clave SSH
ssh-keygen -t ed25519 -C "tu@email.com"
# Pulsa Enter para aceptar ruta por defecto

# 2. Ver la clave pública
cat ~/.ssh/id_ed25519.pub

# 3. Cópiala y pégala en GitHub → Settings → SSH Keys

# 4. Probar conexión
ssh -T git@github.com

Subir un proyecto local a GitHub

# 1. Crea el repositorio vacío en github.com (sin README)

# 2. En tu proyecto local ya inicializado con git init:
git remote add origin git@github.com:usuario/mi-repo.git

# 3. Renombrar rama a main (si era master)
git branch -M main

# 4. Subir
git push -u origin main
# El flag -u enlaza la rama local con la remota (solo la primera vez)

Comandos remotos esenciales

git remote -v              # ver remotos configurados
git push origin main       # subir commits
git pull origin main       # descargar y fusionar cambios
git fetch origin           # descargar sin fusionar
