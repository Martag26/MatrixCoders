Integración con Eclipse: Git y GitHub en Java

EGit — Git integrado en Eclipse

Eclipse incluye EGit, un plugin nativo para trabajar con Git. No necesitas instalar nada adicional si usas Eclipse IDE for Java Developers.

Inicializar un repositorio en Eclipse

  1. Clic derecho en el proyecto → Team → Share Project
  2. Selecciona Git → Create o elige un repositorio existente
  3. Marcar "Use or create repository in parent folder"
  4. Clic en Finish

Commit desde Eclipse

  1. Clic derecho en el proyecto → Team → Commit
  2. En el panel "Git Staging": arrastra archivos de "Unstaged" a "Staged"
  3. Escribe el mensaje de commit
  4. Clic en Commit (solo local) o Commit and Push

Clonar desde GitHub en Eclipse

  1. File → Import → Git → Projects from Git
  2. Clone URI → pega la URL del repo de GitHub
  3. Selecciona la rama y la carpeta de destino
  4. Importa como Existing Eclipse projects

Vista History en Eclipse

Clic derecho en el proyecto → Team → Show in History. Muestra el árbol de commits con autor, fecha, mensaje y archivos modificados en cada snapshot.

Ramas en Eclipse

# Equivalentes en Eclipse:
git branch feature/login  →  Team → Switch To → New Branch
git checkout main         →  Team → Switch To → main
git merge feature/login   →  Team → Merge
💡 Si prefieres la línea de comandos, la terminal integrada en Eclipse (Window → Show View → Terminal) funciona perfectamente con todos los comandos de Git.
