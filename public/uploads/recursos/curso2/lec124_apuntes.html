Apuntes — Introducción a React

Frontend con React · U1 · Fundamentos

React es una librería de JavaScript creada por Meta para construir interfaces de usuario mediante componentes reutilizables. Su filosofía es declarativa: describes cómo debe verse la UI según el estado, y React se encarga de actualizar el DOM.

¿Por qué React?

• Componentes reutilizables → menos duplicación.
• Renderizado declarativo → más fácil de razonar.
• Virtual DOM → actualizaciones eficientes.
• Ecosistema gigantesco (Next.js, React Router, Redux, etc.).

Single Page Application (SPA)

Una aplicación React es típicamente una SPA: una sola página HTML donde el contenido cambia dinámicamente sin recargar. El servidor solo entrega el bundle inicial; el resto lo gestiona JavaScript.

Primer ejemplo

function Saludo() {
  return <h1>¡Hola, React!</h1>;
}

export default Saludo;

Herramientas que necesitas

• Node.js y npm (o pnpm/yarn).
• Un editor (VS Code recomendado).
• Conocimientos previos de HTML, CSS y JavaScript moderno (ES6+).
