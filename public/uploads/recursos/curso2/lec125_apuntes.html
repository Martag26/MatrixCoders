Frontend con React · U1 · JSX

JSX es una extensión de JavaScript que permite escribir marcado parecido a HTML directamente en el código. No es HTML real: se compila a llamadas a React.createElement().

Reglas básicas de JSX

• Las etiquetas se cierran SIEMPRE (incluso <img />, <br />).
• Solo se puede devolver un elemento raíz (envuélvelo en <> </> si necesitas varios).
• class → className, for → htmlFor (porque class y for son palabras reservadas en JS).
• Atributos en camelCase: onclick → onClick, tabindex → tabIndex.

Interpolar JavaScript

const nombre = 'Ana';
const saludo = <p>Hola, {nombre.toUpperCase()}</p>;

Condicionales en JSX

{logueado ? <Dashboard /> : <Login />}

{tieneMensajes && <span>Tienes mensajes</span>}

Listas con .map()

<ul>
  {productos.map(p => (
    <li key={p.id}>{p.nombre}</li>
  ))}
</ul>
