Apuntes — Tipos de datos y variables en React

Frontend con React · U1 · JS moderno aplicado

React se apoya en JavaScript moderno (ES6+). Manejar bien los tipos primitivos, objetos, arrays y la inmutabilidad es fundamental para escribir componentes correctos.

let, const y nunca var

Usa const por defecto. Solo usa let si vas a reasignar. Evita var, su ámbito es confuso.

Destructuring

const { nombre, edad } = usuario;
const [primero, segundo] = lista;

Spread y rest

// Copia y añade propiedades sin mutar:
const nuevoUsuario = { ...usuario, edad: 31 };

// Resto de argumentos:
function sumar(...numeros) {
  return numeros.reduce((a, b) => a + b, 0);
}

Inmutabilidad

En React NO debes mutar el estado directamente. Para modificar un objeto/array crea una copia (spread, map, filter…) y actualiza el estado con la nueva referencia.
