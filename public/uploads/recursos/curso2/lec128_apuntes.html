Apuntes — Estado con useState y hooks

Frontend con React · U2 · Estado

El estado (state) es información local de un componente que cambia con el tiempo y provoca un nuevo renderizado cuando se modifica.

useState

import { useState } from 'react';

function Contador() {
  const [n, setN] = useState(0);
  return (
    <button onClick={() => setN(n + 1)}>
      Has pulsado {n} veces
    </button>
  );
}

Actualizaciones basadas en el valor anterior

setN(prev => prev + 1);  // recomendado si depende del valor previo

useEffect — efectos secundarios

import { useEffect } from "react";

useEffect(() => {
  // se ejecuta tras cada render
  document.title = `Contador: ${n}`;
}, [n]); // solo cuando n cambia

Reglas de los hooks

• Solo se llaman en el nivel superior de un componente (no dentro de if/for/while).
• Solo desde componentes o desde otros hooks.
