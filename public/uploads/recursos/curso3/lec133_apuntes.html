¿Qué es Node.js?

Node.js es un entorno de ejecución JavaScript del lado del servidor construido sobre el motor V8 de Chrome. Permite ejecutar código JavaScript fuera del navegador.

  • Event loop de un solo hilo, no bloqueante
  • Modelo de I/O asíncrono basado en callbacks/Promises
  • NPM (Node Package Manager) con +1,5 millones de paquetes
  • Ideal para APIs REST, microservicios y tiempo real (WebSockets)

Instalación y primer programa

# Verificar instalación
node --version   # v20.x.x

# Crear proyecto
mkdir mi-api && cd mi-api

// hello.js
console.log("Hola desde Node.js");

El Event Loop

Node.js procesa operaciones I/O de forma no bloqueante mediante el event loop:

  1. Call Stack — ejecuta código síncrono
  2. Node APIs — delega tareas asíncronas (fs, http...)
  3. Callback Queue — almacena callbacks listos
  4. Event Loop — mueve callbacks al Call Stack cuando está vacío
setTimeout(() => console.log("Asíncrono"), 0);
console.log("Síncrono");
// → Síncrono
// → Asíncrono
