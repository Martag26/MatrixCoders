Coerción y comparación

// Siempre usar === (comparación estricta)
0 == false    // true  (coerción implícita)
0 === false   // false (tipo diferente)

// Conversiones explícitas
Number("42")      // 42
String(100)       // "100"
Boolean(0)        // false
Boolean("texto")  // true
parseInt("10px")  // 10

Buffer y Streams en Node.js

Node maneja datos binarios con Buffer y opera sobre ellos mediante Streams:

const buf = Buffer.from("Hola", "utf8");
console.log(buf.toString("hex"));   // 486f6c61

// Stream de lectura
const fs = require("fs");
const readable = fs.createReadStream("datos.csv");
readable.on("data", chunk => process.stdout.write(chunk));
readable.on("end", () => console.log("Fin"));
