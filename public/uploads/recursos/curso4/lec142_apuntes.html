¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para describir el contenido y la estructura de una página web. No es un lenguaje de programación — es un lenguaje de marcado: organiza el contenido con etiquetas.

  • Define la estructura del documento (cabeceras, párrafos, listas, imágenes…)
  • Se complementa con CSS (estilos) y JavaScript (interactividad)
  • Lo interpreta el navegador, que lo renderiza visualmente

Estructura mínima de un documento HTML5

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Mi primera página</title>
</head>
<body>
  <h1>Hola, mundo</h1>
  <p>Bienvenido a HTML.</p>
</body>
</html>
  • <!DOCTYPE html> — declara HTML5
  • <html> — raíz del documento (con idioma)
  • <head> — metadatos (no visibles)
  • <body> — contenido visible

Etiquetas, atributos y anidamiento

<a href="https://matrixcoders.com" target="_blank" rel="noopener">
  Visita matrixCoders
</a>
  • Etiqueta: <a>...</a>
  • Atributos: href, target, rel
  • Algunas son autocerradas (void elements): <img>, <br>, <hr>, <input>

Herramientas para empezar

  • Editor: VS Code con extensiones Live Server y Prettier
  • Navegador: Chrome / Firefox con DevTools (F12)
  • Validador: validator.w3.org
