¿Qué es CSS?

CSS (Cascading Style Sheets) describe la presentación visual de un documento HTML: colores, tamaños, posicionamiento, animaciones.

/* Sintaxis básica */
selector {
  propiedad: valor;
  propiedad: valor;
}

h1 {
  color: #6B8F71;
  font-size: 2rem;
  margin-bottom: 1rem;
}

Cómo aplicar CSS

<!-- 1. Hoja externa (RECOMENDADO) -->
<link rel="stylesheet" href="estilos.css">

<!-- 2. Estilo interno -->
<style>
  body { font-family: sans-serif; }
</style>

<!-- 3. Inline (evitar) -->
<p style="color: red">Texto</p>

Especificidad y cascada

Cuando varias reglas afectan al mismo elemento, gana la de mayor especificidad:

  • inline (1000) > id (100) > clase / atributo / pseudoclase (10) > etiqueta (1)
  • En empate, gana la última declarada (orden del código)
  • !important sobreescribe todo — usar solo como último recurso
p             { color: black; }   /* especificidad 1   */
.intro        { color: blue;  }   /* especificidad 10  */
#hero p       { color: green; }   /* especificidad 101 */
[style="..."] { color: red;   }   /* inline = 1000     */
