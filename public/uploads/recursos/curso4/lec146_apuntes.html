El box model

Cada elemento HTML es una caja con 4 capas:

┌───────────────────────────────┐
│         margin (espacio fuera)│
│  ┌─────────────────────────┐  │
│  │   border (borde)        │  │
│  │  ┌───────────────────┐  │  │
│  │  │ padding (relleno) │  │  │
│  │  │  ┌─────────────┐  │  │  │
│  │  │  │  content    │  │  │  │
│  │  │  └─────────────┘  │  │  │
│  │  └───────────────────┘  │  │
│  └─────────────────────────┘  │
└───────────────────────────────┘
/* Recomendado SIEMPRE al inicio */
*, *::before, *::after {
  box-sizing: border-box;
}

.card {
  width: 300px;        /* incluye padding y border */
  padding: 16px;
  border: 1px solid #ddd;
  margin: 0 auto;
}

display: los tipos básicos

  • block — ocupa toda la línea (div, p, section)
  • inline — sólo lo necesario (span, a)
  • inline-block — inline pero acepta width/height
  • none — oculta y no ocupa espacio
  • flex / grid — sistemas de maquetación modernos

Posicionamiento

position: static    /* por defecto */
position: relative  /* offset desde su posición normal */
position: absolute  /* respecto al ancestro positioned */
position: fixed     /* respecto al viewport */
position: sticky    /* híbrido relative/fixed al scroll */
