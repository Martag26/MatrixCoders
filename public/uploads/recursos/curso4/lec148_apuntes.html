Variables CSS (Custom Properties)

:root {
  --color-primario:   #6B8F71;
  --color-fondo:      #f8faf8;
  --color-texto:      #1a2e1a;
  --espaciado-base:   8px;
  --radius:           10px;
  --shadow:           0 2px 8px rgba(0,0,0,.08);
  --font-base:        "Saira", sans-serif;
}

.btn {
  background: var(--color-primario);
  padding: calc(var(--espaciado-base) * 2);
  border-radius: var(--radius);
  box-shadow: var(--shadow);
}

Permite cambiar el tema entero con dos líneas (modo oscuro, marca blanca…).

Responsive design con media queries

/* Mobile-first: estilos base para móvil */
.container { padding: 16px; }

/* Tablet en adelante */
@media (min-width: 768px) {
  .container { padding: 24px; }
}

/* Desktop */
@media (min-width: 1024px) {
  .container { padding: 32px; max-width: 1280px; margin: 0 auto; }
}

Convenciones de naming (BEM)

<div class="card">
  <img class="card__imagen">
  <h3 class="card__titulo">...</h3>
  <button class="card__btn card__btn--primario">Comprar</button>
</div>
  • Bloque: .card
  • Elemento: .card__titulo (parte del bloque)
  • Modificador: .card__btn--primario (variante)
