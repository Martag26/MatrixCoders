¿Qué es JavaScript?

JavaScript (JS) es el lenguaje de programación nativo de los navegadores. Junto a HTML y CSS forma la triada del desarrollo web. Hoy también corre en servidores (Node.js), apps móviles (React Native) y de escritorio (Electron).

  • Lenguaje interpretado y dinámicamente tipado
  • Orientado a objetos basado en prototipos (y con clases desde ES6)
  • Funcional y orientado a eventos
  • Estándar: ECMAScript (ES5, ES6/2015, ES2024…)

Cómo incluir JS en una página

<!-- Inline (evitar) -->
<button onclick="alert('Hola')">Saluda</button>

<!-- Interno -->
<script>
  console.log('Cargado');
</script>

<!-- Externo (RECOMENDADO) -->
<script src="app.js" defer></script>
<script src="modulo.js" type="module"></script>
  • defer — descarga en paralelo, ejecuta tras parsear el HTML (orden conservado)
  • async — descarga en paralelo, ejecuta en cuanto termina (sin orden)
  • type="module" — módulos ES con import/export

Herramientas de navegador

  • Consola (F12 → Console): pruebas rápidas, errores, console.log/warn/error/table
  • Sources: depurar con breakpoints
  • Network: ver peticiones HTTP
  • Application: localStorage, cookies, service workers
