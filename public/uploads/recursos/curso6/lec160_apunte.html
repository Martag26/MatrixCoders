Introducción al lenguaje PHP

¿Qué es PHP?

PHP (Hypertext Preprocessor) es un lenguaje de programación de lado servidor diseñado especialmente para el desarrollo web. A diferencia de HTML, que es estático, PHP genera contenido dinámico que el servidor procesa antes de enviarlo al navegador.

💡 PHP es uno de los lenguajes más usados en la web: WordPress, Facebook (origen) y Wikipedia están construidos sobre él.

¿Cómo funciona?

El flujo es: el navegador hace una petición → el servidor ejecuta el script PHP → se genera HTML puro → se envía al cliente. El usuario nunca ve el código PHP.

Primer script PHP

// archivo: index.php
<?php
// Todo el código PHP va entre estas etiquetas
echo "Hola, mundo desde PHP!";
?>

PHP embebido en HTML

<!DOCTYPE html>
<html>
<body>
  <h1><?php echo "Bienvenido al sitio"; ?></h1>
  <p>Hoy es <?= date('d/m/Y') ?></p>
</body>
</html>

La etiqueta <?= es un atajo para <?php echo.

Comentarios

<?php
// Comentario de una línea
/* Comentario
   multilínea */
# También válido (estilo shell)
?>

Configuración básica con XAMPP

  • Instala XAMPP → inicia Apache
  • Coloca tus archivos en C:/xampp/htdocs/
  • Accede desde http://localhost/tu-archivo.php
  • Consulta errores en C:/xampp/php/logs/
⚠️ Siempre guarda los archivos con extensión .php, no .html, para que el servidor los procese.
