Sintaxis básica de PHP

Variables

En PHP todas las variables comienzan con $ y son de tipado dinámico.

<?php
$nombre  = "Ana";
$edad    = 28;
$precio  = 19.99;
$activo  = true;

echo "Hola, $nombre. Tienes $edad años.";
?>

Estructuras condicionales

<?php
$nota = 7;

if ($nota >= 9) {
    echo "Sobresaliente";
} elseif ($nota >= 5) {
    echo "Aprobado";
} else {
    echo "Suspenso";
}

// Operador ternario
$estado = ($nota >= 5) ? "Aprobado" : "Suspenso";
?>

Bucles

<?php
// for
for ($i = 0; $i < 5; $i++) {
    echo "Vuelta: $i\n";
}

// while
$count = 0;
while ($count < 3) {
    echo $count++;
}

// foreach con array
$frutas = ["manzana", "pera", "uva"];
foreach ($frutas as $fruta) {
    echo "- $fruta\n";
}
?>
💡 Usa foreach para recorrer arrays — es más limpio y seguro que un for con índice.
