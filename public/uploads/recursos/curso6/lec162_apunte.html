Arrays en PHP

<?php
// Array indexado
$colores = ["rojo", "verde", "azul"];
echo $colores[0]; // rojo

// Array asociativo
$persona = [
    "nombre" => "Luis",
    "edad"   => 30,
    "ciudad" => "Madrid"
];
echo $persona["nombre"]; // Luis

// Array multidimensional
$alumnos = [
    ["nombre" => "Ana",  "nota" => 9],
    ["nombre" => "Pedro", "nota" => 7],
];
echo $alumnos[0]["nombre"]; // Ana
?>

Funciones útiles para strings

<?php
$texto = "  Hola Mundo  ";

strlen($texto);          // longitud
trim($texto);           // quitar espacios
strtolower($texto);     // minúsculas
strtoupper($texto);     // mayúsculas
str_replace("Mundo", "PHP", $texto); // reemplazar
explode(",", "a,b,c"); // string → array
implode(",", ["a","b"]); // array → string
?>

Conversión de tipos (casting)

<?php
$num    = (int) "42abc";   // → 42
$texto  = (string) 3.14; // → "3.14"
$logico = (bool) 0;       // → false

// Comprobar tipo
gettype($num);   // "integer"
is_string($num); // false
is_int($num);    // true
?>
💡 Usa siempre === (identidad) en lugar de == (igualdad) para evitar conversiones implícitas inesperadas.
