Funciones y estructuras de control

Definir funciones

<?php
function saludar($nombre) {
    return "Hola, $nombre!";
}

echo saludar("Ana"); // Hola, Ana!
?>

Parámetros con valor por defecto

<?php
function potencia($base, $exp = 2) {
    return pow($base, $exp);
}

echo potencia(3);    // 9  (3²)
echo potencia(2, 10); // 1024
?>

Switch / match

<?php
$dia = 3;

// switch (PHP 4+)
switch ($dia) {
    case 1: echo "Lunes";     break;
    case 2: echo "Martes";    break;
    case 3: echo "Miércoles"; break;
    default: echo "Otro día";
}

// match (PHP 8+) — más estricto y conciso
$nombre = match($dia) {
    1 => "Lunes",
    2 => "Martes",
    3 => "Miércoles",
    default => "Otro"
};
?>

Funciones anónimas y flechas

<?php
// Función anónima (closure)
$doble = function($n) { return $n * 2; };

// Función flecha (PHP 7.4+)
$triple = fn($n) => $n * 3;

// Usadas con array_map
$nums    = [1, 2, 3, 4];
$dobles  = array_map($doble, $nums);
$filtrado = array_filter($nums, fn($n) => $n > 2);
?>
💡 array_map, array_filter y array_reduce son herramientas clave para manipular arrays de forma funcional.

Alcance de variables (scope)

<?php
$global = "soy global";

function prueba() {
    // $global NO es accesible aquí sin declararlo
    global $global; // ahora sí
    echo $global;
}
?>
