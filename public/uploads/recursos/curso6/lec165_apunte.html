Manejo de errores en PHP

Tipos de errores en PHP

TipoDescripción
NoticeVariable no definida — aviso, no fatal
WarningProblema pero el script continúa
Fatal ErrorEl script se detiene
ExceptionError controlado con try/catch

Try / Catch / Finally

<?php
function dividir($a, $b) {
    if ($b === 0) {
        throw new InvalidArgumentException("No se puede dividir entre cero");
    }
    return $a / $b;
}

try {
    $res = dividir(10, 0);
} catch (InvalidArgumentException $e) {
    echo "Error: " . $e->getMessage();
} finally {
    echo "\nEsto siempre se ejecuta.";
}
?>

Excepciones personalizadas

<?php
class DatabaseException extends RuntimeException {}
class NotFoundException extends RuntimeException {}

function buscarUsuario(int $id) {
    // Simulación
    if ($id <= 0) throw new NotFoundException("Usuario no encontrado");
    return ["id" => $id, "nombre" => "Test"];
}

try {
    $user = buscarUsuario(-1);
} catch (NotFoundException $e) {
    // Manejar caso específico
    http_response_code(404);
    echo "404: " . $e->getMessage();
}
?>
