Java

Instalación del JRE y JDK

Descargar el JDK

Para desarrollar con Java necesitas el JDK (Java Development Kit). La versión LTS más usada es Java 17 o Java 21.

  • Descarga oficial: https://adoptium.net (Temurin/Eclipse Foundation)
  • Alternativa Oracle: https://www.oracle.com/java/technologies/downloads/

Verificar la instalación

# En la terminal / símbolo del sistema
java -version
# Debe mostrar algo como: java version "21.0.1"

javac -version
# Debe mostrar: javac 21.0.1

Variable de entorno JAVA_HOME

Muchas herramientas (Maven, Gradle, IDEs) necesitan que JAVA_HOME apunte a la carpeta del JDK.

# Windows (PowerShell)
[System.Environment]::SetEnvironmentVariable("JAVA_HOME","C:\Program Files\Java\jdk-21","Machine")

# Linux / macOS (~/.bashrc o ~/.zshrc)
export JAVA_HOME=/usr/lib/jvm/java-21-openjdk-amd64
export PATH=$JAVA_HOME/bin:$PATH

Compilar y ejecutar desde terminal

# 1. Crear archivo
# HolaMundo.java

# 2. Compilar
javac HolaMundo.java
# Genera HolaMundo.class

# 3. Ejecutar
java HolaMundo
💡 Con Java 11+ puedes ejecutar directamente un archivo fuente sin compilar explícitamente: java HolaMundo.java
