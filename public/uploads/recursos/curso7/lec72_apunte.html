Configuración del IDE Eclipse

¿Qué es Eclipse?

Eclipse es uno de los IDEs más populares para Java. Es gratuito, de código abierto y muy extensible mediante plugins. Alternativas populares son IntelliJ IDEA y VS Code con extensión Java.

Instalación

  1. Descarga Eclipse IDE for Java Developers en https://www.eclipse.org/downloads/
  2. Ejecuta el instalador y selecciona "Eclipse IDE for Java Developers"
  3. Elige la carpeta de instalación y haz clic en "Install"

Crear tu primer proyecto

  1. File → New → Java Project
  2. Escribe el nombre del proyecto (ej: MiPrimerProyecto)
  3. En src, clic derecho → New → Class
  4. Nombre: Main, marca "public static void main"
  5. Clic en Finish

Estructura típica de un proyecto Eclipse

MiProyecto/
├── src/
│   └── com/
│       └── miapp/
│           └── Main.java     // código fuente
├── bin/
│   └── com/
│       └── miapp/
│           └── Main.class    // bytecode compilado
└── .classpath
