Variables locales vs atributos de clase

public class Ejemplo {
    int atributo = 10;  // Atributo de instancia (tiene valor por defecto)

    void metodo() {
        int local = 5; // Variable local (DEBE inicializarse antes de usar)
    }
}
💡 Usa int para enteros normales, double para decimales y boolean para lógica. Los otros tipos son para casos especiales de memoria.
