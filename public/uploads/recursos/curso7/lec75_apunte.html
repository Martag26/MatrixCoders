Java

Uso de constantes (final) y operadores

Constantes con final

final int MAX_INTENTOS = 3;
final double PI = 3.14159265;
final String APP_NAME = "MatrixCoders";

// Un intento de reasignación daría error de compilación:
// MAX_INTENTOS = 5; → ERROR

Operadores básicos

int a = 10, b = 3;

// Aritméticos
a + b    // 13
a - b    // 7
a * b    // 30
a / b    // 3  (división entera!)
a % b    // 1  (módulo/resto)

// División real
(double) a / b   // 3.333...

// Incremento / decremento
a++;  // a = 11
b--;  // b = 2

// Asignación compuesta
a += 5;   // a = a + 5
a *= 2;   // a = a * 2

Operadores de comparación y lógicos

// Comparación → devuelven boolean
a == b    // false
a != b    // true
a >  b    // true
a >= b    // true

// Lógicos
true && false   // false (AND)
true || false   // true  (OR)
!true           // false (NOT)
⚠️ La división entre dos int en Java es siempre entera. Para obtener decimales haz casting: (double) a / b.
