Operadores aritméticos avanzados

Pre-incremento vs Post-incremento

int x = 5;

// Post-incremento: primero usa, luego incrementa
int a = x++;   // a = 5, luego x = 6

// Pre-incremento: primero incrementa, luego usa
int b = ++x;   // x = 7, b = 7

Operador módulo (%)

// Muy útil para:
// 1. Comprobar si un número es par/impar
if (n % 2 == 0) System.out.println("Par");

// 2. Ciclos periódicos
for (int i = 0; i < 20; i++) {
    if (i % 5 == 0) System.out.println("Múltiplo de 5: " + i);
}

// 3. Mantener índice dentro de rango
int idx = (posicion + 1) % array.length;

Conversión de tipos (casting)

// Casting implícito (widening) — automático
int entero = 42;
double decimal = entero;    // OK: 42.0

// Casting explícito (narrowing) — puede perder datos
double pi = 3.14159;
int piEntero = (int) pi;   // 3 (trunca decimales)

// int → String
String texto = String.valueOf(42);       // "42"
String texto2 = Integer.toString(42);   // "42"

// String → int
int num = Integer.parseInt("42");       // 42
💡 Recuerda la jerarquía: byte → short → int → long → float → double. El casting hacia la derecha es automático; hacia la izquierda requiere cast explícito.
