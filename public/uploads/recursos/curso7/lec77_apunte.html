La clase Math I: Potencias, raíces y casting

La clase Math

Java incluye la clase Math en java.lang (no hay que importarla). Todos sus métodos son estáticos.

Métodos esenciales

// Potencias y raíces
Math.pow(2, 10)       // 1024.0  (2^10)
Math.sqrt(144)        // 12.0    (raíz cuadrada)
Math.cbrt(27)         // 3.0     (raíz cúbica)

// Valor absoluto y signo
Math.abs(-15)          // 15

// Máximo y mínimo
Math.max(7, 12)        // 12
Math.min(7, 12)        // 7

// Logaritmos y exponencial
Math.log(Math.E)       // 1.0 (logaritmo natural)
Math.log10(100)       // 2.0 (logaritmo base 10)
Math.exp(1)            // e ≈ 2.718

// Constantes
Math.PI               // 3.14159265358979...
Math.E                // 2.71828182845904...

Ejemplo práctico: calculadora de áreas

double radio = 5.0;
double areaCirculo = Math.PI * Math.pow(radio, 2);
System.out.printf("Área = %.2f%n", areaCirculo); // 78.54

double hipotenusa = Math.sqrt(Math.pow(3, 2) + Math.pow(4, 2));
System.out.println("Hipotenusa = " + hipotenusa); // 5.0
💡 Math.pow(x, 2) y x * x son equivalentes, pero x * x es más rápido para cuadrados.
