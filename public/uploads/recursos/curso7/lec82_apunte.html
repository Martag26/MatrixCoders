Organización de código: Paquetes y la API de Java

¿Qué es un paquete?

Un paquete (package) es una carpeta lógica que agrupa clases relacionadas. Evita conflictos de nombres y organiza el código.

Declarar y usar paquetes

// Archivo: src/com/miapp/modelos/Usuario.java
package com.miapp.modelos;

public class Usuario {
    public String nombre;
}

// Usar desde otra clase
import com.miapp.modelos.Usuario;
import com.miapp.modelos.*;  // importar todo el paquete

Clases wrapper (envolturas)

// Cada tipo primitivo tiene su clase wrapper:
int     ↔  Integer
double  ↔  Double
boolean ↔  Boolean
char    ↔  Character

// Autoboxing (primitivo → wrapper automático)
Integer x = 42;          // boxing
int     y = x;           // unboxing

// Métodos útiles
Integer.parseInt("42")    // → 42
Integer.toBinaryString(10) // → "1010"
Integer.MAX_VALUE         // → 2147483647
💡 Consulta la JavaDoc oficial para ver todos los métodos disponibles en cada clase.
