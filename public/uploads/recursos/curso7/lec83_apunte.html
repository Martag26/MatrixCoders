Entrada de datos con Scanner y JOptionPane

Clase Scanner

import java.util.Scanner;

Scanner sc = new Scanner(System.in);

System.out.print("Introduce tu nombre: ");
String nombre = sc.nextLine();   // Lee una línea completa

System.out.print("Introduce tu edad: ");
int edad = sc.nextInt();         // Lee un entero

System.out.print("Introduce tu altura: ");
double altura = sc.nextDouble(); // Lee un decimal

sc.close();  // Buena práctica: cerrar el scanner

Métodos de Scanner

MétodoLee
nextLine()Línea completa (String)
next()Una palabra (String)
nextInt()Entero
nextDouble()Decimal
nextBoolean()true/false

JOptionPane — ventanas de diálogo

import javax.swing.JOptionPane;

// Leer con ventana de diálogo
String nombre = JOptionPane.showInputDialog("¿Cómo te llamas?");

// Mostrar mensaje
JOptionPane.showMessageDialog(null, "Hola, " + nombre + "!");

// Confirmar (Sí/No)
int resp = JOptionPane.showConfirmDialog(null, "¿Continuar?");
if (resp == JOptionPane.YES_OPTION) { /* ... */ }
⚠️ Después de nextInt() o nextDouble(), llama a sc.nextLine() para consumir el salto de línea sobrante antes de leer un String.
