Bucle while en Java

Sintaxis del while

while (condicion) {
    // se ejecuta mientras condición sea true
}

Ejemplo básico: cuenta atrás

int n = 10;
while (n > 0) {
    System.out.println(n);
    n--;
}
System.out.println("¡Despegue!");

Bucle con validación de entrada

Scanner sc = new Scanner(System.in);
int numero = -1;

while (numero < 1 || numero > 10) {
    System.out.print("Introduce un número entre 1 y 10: ");
    numero = sc.nextInt();
}
System.out.println("Número válido: " + numero);

break y continue

int i = 0;
while (true) {         // bucle infinito
    i++;
    if (i % 2 == 0) continue;  // salta pares
    System.out.println(i);
    if (i >= 9)     break;    // sale del bucle
}
// Imprime: 1 3 5 7 9
⚠️ Asegúrate de que la condición del while eventualmente se vuelva false, o tendrás un bucle infinito que congela el programa.
