Bucle for en Java

Sintaxis del for

for (inicialización; condición; actualización) {
    // cuerpo
}

Ejemplos básicos

// Contar del 1 al 10
for (int i = 1; i <= 10; i++) {
    System.out.println(i);
}

// Cuenta atrás
for (int i = 10; i > 0; i--) {
    System.out.print(i + " ");
}

// De 2 en 2
for (int i = 0; i <= 20; i += 2) {
    System.out.print(i + " ");
}

For anidado — tabla de multiplicar

for (int i = 1; i <= 10; i++) {
    for (int j = 1; j <= 10; j++) {
        System.out.printf("%4d", i * j);
    }
    System.out.println();
}
