Práctica: Validación de email y factoriales

Factorial con for

import java.util.Scanner;

Scanner sc = new Scanner(System.in);
System.out.print("Introduce un número (0-12): ");
int n = sc.nextInt();

long factorial = 1;
for (int i = 2; i <= n; i++) {
    factorial *= i;
}
System.out.println(n + "! = " + factorial);

Validación básica de email

public static boolean esEmailValido(String email) {
    if (email == null || email.isEmpty()) return false;

    int arrobaPosicion = email.indexOf('@');
    if (arrobaPosicion <= 0) return false;

    int puntoPosicion = email.lastIndexOf('.');
    if (puntoPosicion <= arrobaPosicion + 1) return false;
    if (puntoPosicion >= email.length() - 1) return false;

    return true;
}

// Uso
System.out.println(esEmailValido("user@ejemplo.com")); // true
System.out.println(esEmailValido("invalido"));           // false

Validación con regex (más robusto)

String regex = "^[\\w.-]+@[\\w.-]+\\.[a-zA-Z]{2,}$";
boolean valido = email.matches(regex);
💡 Para validaciones simples en ejercicios usa la versión manual. En proyectos reales, usa regex o librerías como Apache Commons Validator.
