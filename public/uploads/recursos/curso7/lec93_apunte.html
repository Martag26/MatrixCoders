Declaración y sintaxis de Arrays

Arrays multidimensionales

// Matriz 3x3
int[][] matriz = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

matriz[1][2]   // 6 (fila 1, columna 2)
matriz.length       // 3 (filas)
matriz[0].length    // 3 (columnas)

// Recorrer matriz
for (int i = 0; i < matriz.length; i++) {
    for (int j = 0; j < matriz[i].length; j++) {
        System.out.printf("%3d", matriz[i][j]);
    }
    System.out.println();
}

Métodos útiles con Arrays

import java.util.Arrays;

int[] nums = {5, 3, 8, 1, 9};

Arrays.sort(nums);              // ordena: [1, 3, 5, 8, 9]
Arrays.toString(nums);          // "[1, 3, 5, 8, 9]"
Arrays.fill(nums, 0);           // [0, 0, 0, 0, 0]
Arrays.copyOf(nums, 3);         // [0, 0, 0]
Arrays.binarySearch(nums, 3);   // índice de 3 (array debe estar ordenado)

Array de objetos

String[] nombres = new String[3];
nombres[0] = "Ana";
nombres[1] = "Luis";
nombres[2] = "María";

Arrays.sort(nombres);   // ["Ana", "Luis", "María"]
💡 Para colecciones de tamaño dinámico (que crecen o menguan), usa ArrayList<T> en lugar de arrays.
