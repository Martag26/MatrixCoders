El bucle for-each en Java

Sintaxis del for-each

El for-each (también llamado enhanced for) simplifica el recorrido de arrays y colecciones. No necesitas índice.

for (tipo variable : coleccion) {
    // usa variable
}

Ejemplos

int[] notas = {8, 6, 9, 7, 10};

// for clásico
for (int i = 0; i < notas.length; i++) {
    System.out.println(notas[i]);
}

// for-each (más limpio)
for (int nota : notas) {
    System.out.println(nota);
}

// Calcular suma con for-each
int suma = 0;
for (int n : notas) suma += n;
System.out.println("Media: " + (double) suma / notas.length);

For-each con ArrayList

import java.util.ArrayList;

ArrayList<String> nombres = new ArrayList<>();
nombres.add("Ana");
nombres.add("Luis");
nombres.add("María");

for (String nombre : nombres) {
    System.out.println("Hola, " + nombre);
}

Limitaciones del for-each

  • No puedes modificar el array/colección mientras lo recorres
  • No tienes acceso al índice
  • Solo avanza hacia adelante

Para esos casos, usa el for clásico con índice.

💡 En Java moderno también puedes usar Streams: Arrays.stream(notas).sum() o nombres.stream().forEach(System.out::println).
