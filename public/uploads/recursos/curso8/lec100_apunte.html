Introducción a JSX: La mezcla de HTML y JavaScript

¿Qué es JSX?

JSX (JavaScript XML) es una extensión de sintaxis que permite escribir código parecido a HTML dentro de JavaScript. Babel lo convierte a llamadas React.createElement().

Sintaxis básica

// JSX
const elemento = <h1>Hola, mundo</h1>;

// Equivale a:
const elemento = React.createElement('h1', null, 'Hola, mundo');

Expresiones JavaScript en JSX

const nombre = 'Ana';
const edad = 25;

return (
  <div>
    <h1>Hola, {nombre}</h1>          // expresión JS entre {}
    <p>Edad: {edad * 2}</p>          // operaciones
    <p>{edad >= 18 ? 'Mayor' : 'Menor'}</p>  // ternario
  </div>
);

Diferencias JSX vs HTML

HTMLJSX
class="btn"className="btn"
for="campo"htmlFor="campo"
<br><br /> (auto-cierre)
onclick="fn()"onClick={fn}
style="color:red"style={{"{{"}}color:'red'{{"}}"}}

Reglas importantes de JSX

  • Cada componente debe retornar un único elemento raíz
  • Usa <>...</> (Fragment) para agrupar sin añadir nodo al DOM
  • Las etiquetas siempre deben cerrarse: <img />, <input />
⚠️ JSX no es HTML. Babel lo transpila a JavaScript puro, por eso no funciona directamente en el navegador sin un bundler.
