¿Qué son los Componentes?

Concepto de componente

Un componente es una pieza independiente y reutilizable de la interfaz. Piensa en ellos como funciones que reciben datos (props) y devuelven JSX (UI).

// Árbol de componentes típico
App
├── Header
│   └── NavBar
├── Main
│   ├── ProductList
│   │   └── ProductCard (x n)
│   └── Sidebar
└── Footer

Tipos de componentes

TipoSintaxisEstado
Funcionalfunction / arrow fnSí (con Hooks)
De claseclass extends ComponentSí (this.state)

En React moderno (2019+) se usan casi exclusivamente componentes funcionales con Hooks.

Componente funcional mínimo

// Saludo.jsx
function Saludo() {
  return <h1>¡Hola desde un componente!</h1>;
}

export default Saludo;

Usar el componente

import Saludo from './Saludo';

function App() {
  return (
    <div>
      <Saludo />  // uso como etiqueta HTML
      <Saludo />  // reutilizable
    </div>
  );
}
💡 Los nombres de componentes siempre empiezan en mayúscula. React usa eso para distinguirlos de las etiquetas HTML nativas.
