Introducción a las Props

¿Qué son las Props?

Las props (properties) son argumentos que pasas a un componente para personalizarlo. Fluyen siempre de padre a hijo y son de solo lectura.

Pasar y recibir props

// Componente hijo
function Saludo({ nombre, edad }) {     // desestructuración
  return (
    <p>Hola, {nombre}. Tienes {edad} años.</p>
  );
}

// Componente padre — pasa props como atributos
function App() {
  return (
    <div>
      <Saludo nombre="Ana" edad={25} />
      <Saludo nombre="Luis" edad={30} />
    </div>
  );
}

Props con valores por defecto

function Boton({ texto = 'Aceptar', color = 'blue' }) {
  return <button style={{ background: color }}>{texto}</button>;
}

La prop especial: children

function Tarjeta({ titulo, children }) {
  return (
    <div className="card">
      <h3>{titulo}</h3>
      {children}   // contenido que ponga el padre entre las etiquetas
    </div>
  );
}

// Uso
<Tarjeta titulo="Mi tarjeta">
  <p>Contenido del padre</p>
</Tarjeta>
⚠️ Nunca modifiques props directamente (props.nombre = 'otro'). Son inmutables. Para cambiar datos usa el estado (useState).
