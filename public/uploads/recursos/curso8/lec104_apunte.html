Tipado de props con PropTypes

¿Qué son PropTypes?

PropTypes es una biblioteca que valida los tipos de las props en tiempo de ejecución. Muestra advertencias en la consola si una prop recibe un tipo incorrecto o si falta una prop requerida.

Instalación

npm install prop-types

Uso básico

import PropTypes from 'prop-types';

function Usuario({ nombre, edad, activo }) {
  return <p>{nombre} ({edad}) — {activo ? 'Activo' : 'Inactivo'}</p>;
}

Usuario.propTypes = {
  nombre: PropTypes.string.isRequired,
  edad:   PropTypes.number.isRequired,
  activo: PropTypes.bool,
};

Usuario.defaultProps = {
  activo: true,
};

Tipos disponibles

PropTypeValidación
PropTypes.stringString
PropTypes.numberNumber
PropTypes.boolBoolean
PropTypes.arrayArray
PropTypes.objectObjeto
PropTypes.funcFunción
PropTypes.nodeCualquier cosa renderable
PropTypes.arrayOf(PropTypes.string)Array de strings
💡 En proyectos con TypeScript, PropTypes no es necesario — TypeScript hace la validación en tiempo de compilación de forma más completa.
