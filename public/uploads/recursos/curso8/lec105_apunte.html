CSS en React

Opciones para aplicar estilos

1. CSS global (archivo .css)

/* App.css */
.boton { background: #6B8F71; color: white; padding: 8px 16px; }

// App.jsx
import './App.css';
return <button className="boton">Clic</button>;

2. CSS Modules (scoped)

/* Boton.module.css */
.boton { background: green; }

// Boton.jsx
import styles from './Boton.module.css';
return <button className={styles.boton}>Clic</button>;

3. Estilos en línea (inline)

const estilos = { background: '#6B8F71', color: 'white', borderRadius: '8px' };
return <button style={estilos}>Clic</button>;

4. Styled Components (biblioteca externa)

// npm install styled-components
import styled from 'styled-components';

const Boton = styled.button`
  background: #6B8F71;
  color: white;
  padding: 8px 16px;
`;

return <Boton>Clic</Boton>;
💡 Para proyectos pequeños usa CSS global o inline. Para apps grandes, CSS Modules o Tailwind CSS son las opciones más populares.
