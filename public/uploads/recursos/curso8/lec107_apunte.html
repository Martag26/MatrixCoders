Manejo de eventos en React

Eventos en React vs HTML

HTMLReact (JSX)
onclick="fn()"onClick={fn}
onchange="fn()"onChange={fn}
onsubmit="fn()"onSubmit={fn}
onkeydown="fn()"onKeyDown={fn}

Manejadores de eventos

function Contador() {
  const handleClick = () => {
    alert('¡Botón pulsado!');
  };

  return <button onClick={handleClick}>Pulsa aquí</button>;
}

El objeto event

const handleChange = (e) => {
  console.log(e.target.value);  // valor del input
  console.log(e.target.name);   // nombre del campo
};

return <input type="text" onChange={handleChange} />;

Pasar argumentos a un handler

// Mal: se ejecuta inmediatamente
<button onClick={eliminar(id)}>X</button>

// Bien: arrow function que envuelve la llamada
<button onClick={() => eliminar(id)}>X</button>
💡 Nombra los handlers con el prefijo handle: handleClick, handleSubmit, handleChange. Hace el código más legible.
