React

Actualización del estado y renderizado

Actualización basada en el valor anterior

// Forma simple (puede ser stale)
setCount(count + 1);

// Forma funcional (siempre usa el valor más reciente)
setCount(prev => prev + 1);

Actualizar objetos en el estado

const [usuario, setUsuario] = useState({ nombre: 'Ana', edad: 25 });

// Mal: mutación directa
usuario.edad = 26;  // ¡no provoca re-render!

// Bien: spread operator para crear nuevo objeto
setUsuario({ ...usuario, edad: 26 });

Actualizar arrays en el estado

const [items, setItems] = useState(['a', 'b']);

// Añadir
setItems([...items, 'c']);

// Eliminar por índice
setItems(items.filter((_, i) => i !== 1));

// Modificar un elemento
setItems(items.map((item, i) => i === 0 ? 'NUEVO' : item));
💡 React agrupa múltiples setState en un solo re-render cuando están en el mismo manejador de evento. Esto se llama batching y mejora el rendimiento.
