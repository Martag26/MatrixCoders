Comunicación entre componentes

Flujo de datos unidireccional

En React los datos fluyen hacia abajo (padre → hijo) a través de props. Para que un hijo comunique algo al padre, el padre le pasa una función como prop.

Patrón: elevación de estado (lifting state up)

// Padre: tiene el estado y la función
function Padre() {
  const [mensaje, setMensaje] = useState('');

  return (
    <div>
      <Hijo onMensaje={setMensaje} />
      <p>Recibido: {mensaje}</p>
    </div>
  );
}

// Hijo: llama a la función del padre
function Hijo({ onMensaje }) {
  return (
    <button onClick={() => onMensaje('Hola desde el hijo')}>
      Enviar al padre
    </button>
  );
}

Pasar estado entre hermanos

Si dos componentes hermanos necesitan compartir estado, eleva ese estado al ancestro común más próximo y pásalo como props a ambos.

// AncestroComun tiene el estado compartido
function App() {
  const [valor, setValor] = useState(0);
  return (
    <>
      <HermanoA onCambio={setValor} />
      <HermanoB valor={valor} />
    </>
  );
}
💡 Cuando la comunicación entre componentes se vuelve compleja, considera usar Context API o Redux para manejar estado global.
