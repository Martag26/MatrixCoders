El Hook useEffect: Introducción y sintaxis

¿Qué es useEffect?

useEffect permite ejecutar código después de que React renderice el componente. Se usa para efectos secundarios: llamadas a APIs, suscripciones, timers, manipulación del DOM…

Sintaxis

import { useEffect } from 'react';

useEffect(() => {
  // código del efecto
  return () => {
    // limpieza (opcional)
  };
}, [dependencias]);

Tres comportamientos según dependencias

// 1. Sin array → ejecuta después de CADA render
useEffect(() => { console.log('render!'); });

// 2. Array vacío → ejecuta SOLO al montar el componente (1 vez)
useEffect(() => {
  console.log('montado');
  // ideal para cargar datos iniciales
}, []);

// 3. Con dependencias → ejecuta cuando cambia alguna dep.
useEffect(() => {
  console.log('userId cambió:', userId);
}, [userId]);

Ejemplo: título del documento

const [count, setCount] = useState(0);

useEffect(() => {
  document.title = `Conteo: ${count}`;
}, [count]);
💡 Piensa en useEffect como una versión funcional de los métodos de ciclo de vida de las clases: componentDidMount, componentDidUpdate y componentWillUnmount.
