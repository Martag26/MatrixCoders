React

useEffect: Manejo de dependencias

Array de dependencias

React compara las dependencias con sus valores anteriores. Si alguna cambió, re-ejecuta el efecto.

const [busqueda, setBusqueda] = useState('');
const [resultados, setResultados] = useState([]);

useEffect(() => {
  if (!busqueda) { setResultados([]); return; }

  // Solo busca cuando cambia 'busqueda'
  fetch(`/api/buscar?q=${busqueda}`)
    .then(r => r.json())
    .then(data => setResultados(data));
}, [busqueda]);  // ← dependencia

Reglas de los Hooks

  • Solo llama Hooks en el nivel superior (no dentro de if, loops, funciones)
  • Solo llama Hooks desde componentes funcionales de React o custom hooks
  • El plugin ESLint eslint-plugin-react-hooks te avisa si los incumples

Problema: bucle infinito

// ❌ MAL: objeto en dependencias crea bucle infinito
useEffect(() => {
  fetchData(opciones);
}, [opciones]);  // nuevo objeto en cada render

// ✅ BIEN: usa useMemo o primitivos en dependencias
useEffect(() => {
  fetchData({ page: pagina });
}, [pagina]);
⚠️ Si incluyes una función en el array de dependencias, usa useCallback para memorizar la referencia y evitar re-renders innecesarios.
