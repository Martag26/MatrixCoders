Uso de múltiples Hooks en un mismo componente

Combinar useState y useEffect

import { useState, useEffect } from 'react';

function RelojDigital() {
  const [hora, setHora] = useState(new Date());

  useEffect(() => {
    const id = setInterval(() => {
      setHora(new Date());
    }, 1000);
    return () => clearInterval(id);
  }, []);

  return <h1>{hora.toLocaleTimeString()}</h1>;
}

Múltiples estados independientes

function Perfil() {

Orden de los Hooks

React identifica cada Hook por su orden de llamada. Por eso nunca pongas Hooks dentro de condicionales o bucles.

💡 Si un componente tiene demasiados Hooks, es señal de que deberías extraer lógica a Custom Hooks para mantenerlo limpio.
