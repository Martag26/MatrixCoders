Introducción a useRef: Acceso directo al DOM

¿Qué es useRef?

useRef devuelve un objeto mutable con una propiedad .current. Tiene dos usos principales:

  1. Acceder directamente a un elemento DOM
  2. Guardar un valor que persiste entre renders sin causar re-render

Acceder al DOM

import { useRef } from 'react';

function FocusInput() {
  const inputRef = useRef(null);

  const handleClick = () => {
    inputRef.current.focus();
  };

  return (
    <>
      <input ref={inputRef} type="text" />
      <button onClick={handleClick}>Enfocar</button>
    </>
  );
}

Guardar valores sin re-render

function Cronometro() {
  const [tiempo, setTiempo] = useState(0);
  const intervalRef = useRef(null);

  const iniciar = () => {
    intervalRef.current = setInterval(() => {
      setTiempo(t => t + 1);
    }, 1000);
  };

  const detener = () => clearInterval(intervalRef.current);

  return (
    <>
      <p>{tiempo}s</p>
      <button onClick={iniciar}>Iniciar</button>
      <button onClick={detener}>Detener</button>
    </>
  );
}
⚠️ No uses useRef para leer/guardar datos que deben mostrarse en la UI. Para eso usa useState. useRef es para efectos sin render.
