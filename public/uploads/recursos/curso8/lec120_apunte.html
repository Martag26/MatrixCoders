Consumo de APIs externas con fetch

La API fetch

fetch es la función nativa del navegador para hacer peticiones HTTP. Devuelve una Promise.

GET básico

fetch('https://jsonplaceholder.typicode.com/posts/1')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(err => console.error('Error:', err));

Con async/await (más legible)

const cargarPost = async (id) => {
  try {
    const res  = await fetch(`https://jsonplaceholder.typicode.com/posts/${id}`);
    if (!res.ok) throw new Error(`HTTP ${res.status}`);
    const data = await res.json();
    return data;
  } catch (e) {
    console.error(e);
  }
};

En un componente React

function Post({ id }) {
  const [post, setPost] = useState(null);

  useEffect(() => {
    fetch(`https://jsonplaceholder.typicode.com/posts/${id}`)
      .then(r => r.json())
      .then(data => setPost(data));
  }, [id]);

  if (!post) return <p>Cargando...</p>;
  return <h2>{post.title}</h2>;
}
💡 Para apps más complejas considera usar Axios (mejor manejo de errores) o TanStack Query (caché, revalidación automática).
