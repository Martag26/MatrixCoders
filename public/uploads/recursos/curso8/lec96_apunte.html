React

Introducción a React y ¿por qué usarlo?

¿Qué es React?

React es una biblioteca JavaScript creada por Meta (Facebook) en 2013 para construir interfaces de usuario. No es un framework completo, sino una capa de UI (View) que puedes combinar con otras herramientas.

¿Por qué React?

  • Componentes reutilizables: divide la UI en piezas pequeñas e independientes
  • Virtual DOM: React calcula el mínimo cambio necesario en el DOM real, haciendo las actualizaciones muy eficientes
  • Ecosistema enorme: React Router, Redux, Next.js, React Native…
  • Demanda laboral: es la biblioteca frontend más demandada del mercado
  • Unidireccional: el flujo de datos es predecible (padre → hijo)

El Virtual DOM

Cuando el estado cambia, React crea una copia virtual del DOM, la compara con la anterior (diffing) y solo actualiza los nodos que cambiaron (reconciliation). Esto es mucho más rápido que manipular el DOM directamente.

💡 React no reemplaza HTML ni CSS. Los sigue usando, pero los organiza en componentes reutilizables con lógica JavaScript.
