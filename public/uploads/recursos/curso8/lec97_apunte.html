React

Instalación de herramientas: Node.js y VS Code

Prerequisitos

Para desarrollar con React necesitas:

  1. Node.js (incluye npm) — https://nodejs.org → descarga la versión LTS
  2. VS Code — https://code.visualstudio.com

Verificar instalación

# En la terminal
node --version    # v20.x.x o superior
npm --version     # 10.x.x o superior

Extensiones de VS Code recomendadas

ExtensiónPara qué sirve
ES7+ React SnippetsAtajos: rafce → componente funcional
PrettierFormatea el código automáticamente
ESLintDetecta errores de sintaxis en tiempo real
Auto Rename TagRenombra etiquetas JSX a la vez
GitLensHistorial de cambios git en el editor

npm vs npx

# npm → instala paquetes
npm install react

# npx → ejecuta paquetes sin instalarlos globalmente
npx create-react-app mi-app
💡 Configura Prettier para que formatee al guardar: Settings → Format On Save → On. Tu código siempre quedará limpio.
