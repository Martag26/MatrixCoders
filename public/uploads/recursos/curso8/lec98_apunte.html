Creación del primer proyecto con create-react-app

Crear el proyecto

# En la terminal, en la carpeta donde quieras el proyecto
npx create-react-app mi-primera-app
cd mi-primera-app
npm start

El navegador abrirá http://localhost:3000 con la app en marcha. Cada cambio que guardes se refleja automáticamente (hot reload).

Scripts disponibles

ComandoAcción
npm startServidor de desarrollo en localhost:3000
npm run buildGenera versión optimizada para producción en /build
npm testEjecuta los tests
npm run ejectExpone config de Webpack (irreversible)

Alternativa moderna: Vite

# Más rápido que CRA
npm create vite@latest mi-app -- --template react
cd mi-app
npm install
npm run dev
💡 En proyectos nuevos se recomienda Vite sobre create-react-app. CRA está en mantenimiento reducido desde 2023.
