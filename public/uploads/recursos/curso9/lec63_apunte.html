Introducción y Fundamentos de Node.js

¿Qué es Node.js?

Node.js es un entorno de ejecución de JavaScript en el servidor, construido sobre el motor V8 de Chrome. Permite usar JavaScript fuera del navegador para crear servidores, scripts, herramientas de línea de comandos y aplicaciones de tiempo real.

Características clave

  • Asíncrono y no bloqueante: usa un modelo de I/O basado en eventos, ideal para operaciones de red y disco
  • Single-threaded con event loop: un solo hilo maneja miles de conexiones concurrentes
  • npm: el gestor de paquetes más grande del mundo (>2 millones de paquetes)
  • Same language: JavaScript tanto en frontend como en backend

Event Loop — cómo funciona

// Operación bloqueante (evitar en Node)
const data = fs.readFileSync('file.txt');   // bloquea todo

// Operación no bloqueante (patrón Node)
fs.readFile('file.txt', 'utf8', (err, data) => {
  if (err) throw err;
  console.log(data);
});
console.log('Esto se ejecuta antes que el callback');

Primer servidor HTTP

const http = require('http');

const server = http.createServer((req, res) => {
  res.writeHead(200, { 'Content-Type': 'text/plain' });
  res.end('¡Hola desde Node.js!');
});

server.listen(3000, () => {
  console.log('Servidor en http://localhost:3000');
});

Módulos CommonJS vs ES Modules

// CommonJS (require) — por defecto en Node
const fs = require('fs');
module.exports = { miFuncion };

// ES Modules (import/export) — con "type":"module" en package.json
import fs from 'fs';
export { miFuncion };
